exactComplete :: String -> [String] -> [Completion] Source #

Constructs a list of Completions from a query and completion options by filtering them for prefix matches. A completion will be selected if it's an exact match for a provided option.

completeWithinQueryNamespace :: String -> [String] -> [Completion] Source #

Completes a list of options, limiting options to the same namespace as the query, or the namespace's children if the query is itself a namespace.

E.g. query: "base" would match: ["base", "base.List", "base2"] wouldn't match: ["base.List.map", "contrib", "base2.List"]
